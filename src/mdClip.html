Help

mdClip uses the active tab's title, favicon, meta description, and current selection to create a markdown snippet which is (optionally) copied to the clipboard so it can be pasted into another application.

Clips may also optionally be stored locally so they can be accumulated and then downloaded at a later time. This can be useful if you want to remember pages you come across but don't want to write them out to your notes application each time. When you are ready, use the Storage button and then Export a zip file containing a group of clips.
